=> Present Simple
+ Positive Form:
    - Subject + do/does + v1 + object ....
+ Negative Form:
    - Subject + don't / doesn't + v1 + object ....

+ Question Form:
    + Positive Form:
        - Do/Does + subject + v1 + object?
    + Negative Form:
        - Don't/Doesn't + subject + v1 + object?

=> Present Continuous / Progressive.
+ Positive Form:
    - Subject + be + ing + object.
